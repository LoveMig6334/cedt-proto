Feature Impleamntation list:

    1. หน้า Login
    - ตรวจสอบว่าในขณะนี้ระบบเก็บข้อมูลอีเมลและรหัสผ่านสำหรับ login ไว้ในไหน และทำให้หน้าเว็บ Demo สามารถ login แบบมั่วๆไปก่อนได้ในตอนนี้

    หน้าตั้งค่าโรงงาน:
        หน้า 1:
        - ทำให้ประเภทโรงงาน สามารถเลือกได้ตั้งแต่แปรรูปเนื้อวัวแปรรูปเนื้อไก่เนื้อปลาเนื้อหมู
        - ทำให้ที่ตั้งโรงงาน Link กับ google Map
        - ทำให้ Map อยู่นไทยเท่านั้น

        หน้า 2:
        - เปลี่ยนการตั้งค่าผู้จัดการโรงงานจะให้ใส่ชื่อนามสกุลเป็นใส่ username ในระบบแทน
        - ตรงเพิ่มพนักงาน เปลี่ยนชื่อนามสกุลเป็น username ในระบบ
        - ตรงเพิ่มพนักงาน ลบแผนกออก

        หน้า 3:
        - ลบหน้าเลือกผู้จัดส่งวัตถุดิบออกเพราะ ไม่ได้ fix เป็นเจ้านี้ตลอดเวลาตลอดเวลา และมีฟิวเจอร์นี้อยู่ใน dashboard อยู่แล้ว

        หน้า 4:
        - ลบหน้าที่ 4 ฟีเจอร์นำเข้าข้อมูลออก

    2. หน้า Dashboard

        2.1 หน้าแดชบอร์ดภาพรวม

        2.2 หน้าจัดหาวัตถุดิบ
            - ทำฟีเจอร์ AI Assistant ให้เชื่อมกับ ui ในหน้าจัดหาวัตถุดิบ โดยเมื่อพิมพ์ข้อความให้ AI แล้ว AI จะคัดเลือก 3 Choice ที่เหมาะสมที่สุดมาให้ผู้ใช้เห็น
            - เพิ่มตัวเลือก mock up supplier ในปัจจุบันให้มีประมาณ 10 เจ้า
            - หลังจากกดเลือกเข้าไปยัง supplier ที่ต้องการแล้วผู้ใช้จะสามารถดูข้อมูลรายละเอียดต่างๆเช่นที่ตั้งประเภทเนื้อที่ขายราคาต่อกิโล
            - ออก PO

            - หลังระบ

        2.3 หน้ารับเข้าวัตถุดิบ

        2.4 หน้าควบคุมการผลิต

        2.5 หน้าตรวจสอบคุณภาพ

        2.6 หน้าคลังสินค้า

        2.7 หน้าฟีเจอร์ขายและจัดส่ง

    - ช่วยทำฟีเจอร์ชมตัวอย่างระบบที่มีอยู่ในหน้าแรกหน้าแลนดิ้งเพจของเว็บ Application Face Pro โดยอ้างอิงรายละเอียดของฟีเจอร์ทั้งหมดที่จะอยู่ใน Animation แสดงตัวอย่างระบบนี้ตามสไลด์ที่ผมได้ส่งไปให้หน่อยฟีเจอร์จะครอบคลุมตั้งแต่ขั้นตอนการจัดหาวัตถุดิบไปจนถึงขั้นตอนการส่งออกวัตถุดิบซึ่งอนิเมชั่นทั้งหมดจะถูกแสดงออกมาเป็นแนวเว็บ swg Animation ที่เลื่อนต่อกันไปเรื่อยๆแบบอัตโนมัติและมีปุ่ม Post ให้ผู้ใช้สามารถกดได้ด้านล่าง
