4. ที่มาและความสำคัญ
ในปัจจุบัน อุตสาหกรรมการแปรรูปเนื้อสัตว์เป็นหนึ่งในอุตสาหกรรมที่มีความสำคัญต่อเศรษฐกิจของประเทศ และมีบทบาทสำคัญในการบริโภคภายในประเทศและการส่งออกในระดับโลก โดยในปี 2568 มีผู้ประกอบการธุรกิจแปรรูปเนื้อสัตว์ในประเทศไทยจำนวน 9,488 ราย แบ่งเป็นบุคคลธรรมดา 8,050 ราย นิติบุคคล 1,011 ราย และวิสาหกิจชุมชน 427 ราย และมีแนวโม้มยอดขายขยายตัวอย่างต่อเนื่องที่ร้อยละ 4.3
อย่างไรก็ตาม อุตสาหกรรมการแปรรูปเนื้อสัตว์ยังมีปัญหาในด้านการบริหารจัดการ เนื่องจากกระบวนการในโรงงานอุตสาหกรรมการแปรรูปเนื้อสัตว์มีความซับซ้อน มีหลายขั้นตอนในการดำเนินงาน ตั้งแต่การจัดหาวัตถุดิบ การคัดแยก การแปรรูป การจัดเก็บ ไปจนถึงการกระจายสินค้า แม้ว่าจะมีการใช้เทคโนโลยีต่างๆ ได้แก่ การนำเครื่องจักรและการจัดการข้อมูลในระบบดิจิทัล เข้ามาช่วยลดภาระในการทำงาน ทำให้กระบวนการทั้งหมดใช้เวลาน้อยลง และลดต้นทุนได้มาก แต่ก็ยังพบปัญหาในด้านการบริหารจัดการนี้อยู่
ด้วยการพัฒนาของเทคโนโลยี AI ในปัจจุบัน ทำให้สามารถจัดการและวิเคราะห์ข้อมูลเชิงปฏิบัติการได้อย่างมีประสิทธิภาพ เราจึงมองเห็นถึงโอกาสในการแก้ปัญหาและต้องการที่จะพัฒนาโปรเจกต์นี้ขึ้นมาเพื่อเพิ่มประสิทธิภาพในกระบวนการทำงานของอุตสาหกรรมการแปรรูปเนื้อสัตว์ โดยออกแบบให้มีความง่ายในการนำไปประสานกับกระบวนการผลิตที่มีอยู่และทำให้เกิดกระทบกับการทำงานของพนักงานและระบบที่มีอยู่ให้น้อยที่สุด
โดยได้เลือกโรงงานแปรรูปเนื้อวัวมาใช้เป็นต้นแบบในการพัฒนาเว็บแอปพลิเคชันในระยะเริ่มต้น ซึ่งได้ดำเนินการสัมภาษณ์ผู้ประกอบการโรงงานแปรรูปเนื้อวัว เพื่อทราบถึงปัญหาและความต้องการของอุตสาหกรรม นำมาใช้เป็นแนวทางในการพัฒนาโปรเจกต์ให้เหมาะสมกับสภาพการใช้งานจริงได้อย่างมีประสิทธิภาพมากที่สุด

8. การทบทวนวรรณกรรม (Literature Review)
ในการพัฒนาโครงงานนี้ ผู้จัดทำได้ศึกษาหลักการและทฤษฎีที่เกี่ยวข้อง ดังนี้
การเรียนรู้เชิงลึก (Deep Learning) และ ResNet-18 ศึกษาการทำงานของ Convolutional Neural Networks (CNN) โดยเลือกใช้สถาปัตยกรรม ResNet-18 ซึ่งมีความเหมาะสมกับการประมวลผลบนอุปกรณ์ Embedded System อย่าง KidBright µAI เนื่องจากมีความแม่นยำสูงและใช้ทรัพยากรไม่มากจนเกินไป
อินเทอร์เน็ตของสรรพสิ่ง (Internet of Things - IoT) ศึกษาโปรโตคอล MQTT (Message Queuing Telemetry Transport) ซึ่งเป็นมาตรฐานการสื่อสารของอุปกรณ์ IoT ที่มีขนาดเล็ก ใช้แบนด์วิดท์น้อย และรองรับการทำงานแบบ Publish/Subscribe ผ่าน WebSocket เพื่อเชื่อมต่อกับ Web Browser 
จิตวิทยาพฤติกรรมและ Gamification ศึกษาแนวคิดการให้รางวัล (Reward System) เพื่อปรับเปลี่ยนพฤติกรรมมนุษย์ โดยการนำระบบแต้ม (Points) และการตอบคำถาม (Quiz) มากระตุ้นให้ผู้ใช้เกิดความสนใจและจดจำวิธีการแยกขยะได้ดียิ่งขึ้น

9. สรุปผลงานนวัตกรรม
แพลตฟอร์มบริหารจัดการอุตสาหกรรมการแปรรูปเนื้อสัตว์ มีชื่อว่า FreshPro
โดยมีฟีเจอร์ดังนี่
Features จัดหาวัตถุดิบ
ฟีเจอร์จัดหาวัตถุดิบ ทำให้ผู้ใช้สามารถค้นหา supplier หรือแหล่งจำหน่ายวัตถุดิบได้ตามความต้องการ
ฟีเจอร์นี้ยังรองรับระบบการแนะนำ ด้วย AI ที่จะช่วยแนะนำ supplier ให้ตรงตามความต้องการและคุ้มค่าที่สุดและหลังจากเลือก supplier ที่ต้องการแล้ว ฟีเจอร์นี้ยังอำนวยความสะดวกด้วยการออกใบสั่งซื้อ เพื่อให้ฝ่ายจัดซื้อดำเนินการต่อไปได้ง่ายขึ้น
Features รับเข้าวัตถุดิบ
ฟีเจอร์รับเข้าวัตถุดิบ จะนำข้อมูลการสั่งซื้อวัตถุดิบของโรงงาน มาแสดงผลพร้อมกับวันที่รับของ เพื่อแจ้งเตือนให้พนักงานรับทราบ
เมื่อวัตถุดิบมาถึงโรงงานแล้ว พนักงานจะบันทึกข้อมูลการรับเข้าลงระบบ และถ่ายรูปภาพยืนยัน
ข้อมูลที่บันทึกจะนำไปวิเคราะห์เป็นยอดการสั่งซื้อ รายวัน รายสัปดาห์ รายเดือน และรายปีได้
Features ควบคุมการผลิต
ฟีเจอร์ควบคุมการผลิต มีหน้าที่ติดตามการผลิตในแต่ละล็อต และเก็บข้อมูลผลิตภัณฑ์ที่ได้ รวมถึงของเสียที่เกิดขึ้นในกระบวนการลงระบบ\
ข้อมูลที่บันทึกจะนำไปวิเคราะห์ประสิทธิภาพการผลิตเป็น รายวัน รายสัปดาห์ รายเดือน และรายปีได้
บันทึกข้อมูลการเบิกของจากคลัง และแสดงข้อมูลได้แบบ Real Time
Features ตรวจสอบคุณภาพ AI
ฟีเจอร์นี้ใช้เพื่อตรวจสอบคุณภาพของเนื้อในแต่ละชิ้น โดยตรวจสอบสีของเนื้อ และความสมบูรณ์ของรูปทรง โดยให้ Ai ตรวจสอบจากการรับภาพผ่านกล้อง และให้คะแนนโดยมีเกณฑ์มาตรฐานที่โรงงานกำหนดไว้
ข้อมูลทั้งหมดจะถูกบันทึกไว้และสามารถนำไปวิเคราะห์คุณภาพการผลิตในระยะยาวต่อได้
Features คลังสินค้า
ฟีเจอร์คลังสินค้าจะบันทึกข้อมูลต่อจากขั้นตอนการตรวจสอบคุณภาพ สินค้าในแต่ละล็อตจะถูกบันทึกสถานะต่างๆ และมีการคำนวณวันหมดอายุเพื่อแจ้งเตือนผู้ใช้
สินค้าที่อยู่ในคลังจะอยู่ในสถานะรอ match ผู้ซื้อ แล้วเมื่อมีการส่งออกไปขายข้อมูลก็จะถูกบันทึกไว้ด้วยและบันทึกข้อมูลหลังส่งไปขาย เพื่อเก็บยอดไว้วิเคราะห์อีกที
Features ขาย & จัดส่ง
ฟีเจอร์ขายและจัดส่ง สามารถจัดรายการสินค้าตามออเดอร์ของลูกค้าได้
และบันทึกข้อมูลการจัดส่งและยอดขายเป็นรายวัน รายสัปดาห์ รายเดือน และรายปี เพื่อนำไปวิเคราห์
