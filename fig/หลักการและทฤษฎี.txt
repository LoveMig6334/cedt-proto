สถาปัตยกรรม Frontend แบบ Component-Based (Component-Based UI Architecture) ออกแบบส่วนติดต่อผู้ใช้งาน โดยแบ่งออกเป็นหน่วยย่อยที่สามารถนำกลับมาใช้ซ้ำได้ โดยใช้ไลบรารี React 19 เป็นหลัก ทำให้แต่ละส่วนของระบบ เช่น Sidebar, Topbar, แบบฟอร์มรับวัตถุดิบ และตารางผลการตรวจสอบ แยกออกจากกันอย่างชัดเจน ง่ายต่อการพัฒนาและดูแลรักษาระบบ

สถาปัตยกรรม Backend แบบ RESTful API ด้วย Python และ Flask ออกแบบตามแนวคิด REST (Representational State Transfer) ซึ่งแบ่งการทำงานออกเป็นชั้น ๆ อย่างชัดเจน โดยใช้ Python เป็นภาษาหลักเนื่องจากมีไลบรารีสำหรับ Data Science และ Machine Learning  และใช้ Flask เป็น Micro Web Framework สำหรับสร้าง API Endpoint ที่มีน้ำหนักเบาและยืดหยุ่น ทำหน้าที่เป็นตัวกลางเชื่อมต่อระหว่าง 3 ส่วนหลักของ Backend ได้แก่ โมเดล Deep Learning (YOLO) สำหรับประมวลผลภาพตรวจสอบคุณภาพ, Google Gemini API สำหรับสรุปข้อมูลด้วย Gemini AI และฐานข้อมูล Supabase โดย Flask จะรับคำขอจากเว็บแอปพลิเคชัน Next.js ส่งต่อไปยังโมเดล AI แล้วส่งผลลัพธ์กลับในรูปแบบ JSON เพื่อนำไปแสดงผลบนหน้าจอ

ใช้ระบบฐานข้อมูลด้วย Supabase ซึ่งเป็นแพลตที่ให้บริการฐานข้อมูล PostgreSQL พร้อม REST API และระบบยืนยันตัวตนในตัว เพื่อจัดเก็บข้อมูลหลักของระบบ เช่น ข้อมูลผู้ใช้งาน ข้อมูลวัตถุดิบ และข้อมูลการผลิต โดยเชื่อมโยงกับ Python Flask API ซึ่งทำหน้าที่ประมวลผลและวิเคราะห์ข้อมูลเชิงสถิติก่อนส่งผลลัพธ์กลับไปแสดงบนเว็บแอปพลิเคชัน

การวิเคราะข้อมูล ด้วยโมเดลภาษาขนาดใหญ่ (Large Language Model — LLM) โดยเลือกใช้ Google Gemini API ซึ่งรองรับการรับข้อมูลตัวเลขและข้อความจากระบบและสรุปผลออกมาในรูปแบบภาษาธรรมชาติ นำมาใช้ในระบบ FreshPro เพื่อวิเคราะห์และสรุปข้อมูล เช่น ยอดขายรายช่วง ผลผลิตสะสม ประสิทธิภาพสายการผลิต และแนวโน้มของเสีย ให้ออกมาในรูปแบบที่ผู้บริหารสามารถอ่านและนำไปประกอบการตัดสินใจได้ทันที โดยไม่จำเป็นต้องวิเคราะห์ข้อมูลดิบด้วยตนเอง

การทำ Classification model โดยใช้โมเดลการทำงานของโครงข่ายประสาทเทียมแบบ Convolutional Neural Networks (CNN) โดยเลือกใช้สถาปัตยกรรม YOLO (You Only Look Once) ในโหมด Classification ซึ่งได้รับการออกแบบมาเพื่อจำแนกประเภทของวัตถุจากภาพแบบ Real-time ทำให้เหมาะสมกับการใช้งานในสายการผลิตที่ต้องการผลลัพธ์ทันที จึงนำมาประยุกต์ใช้ในฟีเจอร์ตรวจสอบคุณภาพ AI ของระบบ FreshPro เพื่อวิเคราะห์สีและความสมบูรณ์ของรูปทรงเนื้อแต่ละชิ้น และจำแนกเกรดตามเกณฑ์ที่โรงงานกำหนด (เกรด A–D) โดยบันทึกผลลัพธ์ทั้งหมดไว้เพื่อวิเคราะห์แนวโน้มคุณภาพการผลิตในระยะยาว
